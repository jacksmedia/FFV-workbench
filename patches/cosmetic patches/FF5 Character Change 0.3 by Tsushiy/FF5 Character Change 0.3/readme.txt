
		/_/_/_/ Final Fantasy V キャラクター画像変更パッチ /_/_/_/


　　この度は本パッチをダウンロードして頂き、誠にありがとうございます。
　このパッチを使用するにはフリーソフト「WinIPS　SMB Laboratory」(http://smblabo.web.fc2.com/)
　が必要です。
　　また、このパッチは、ヘッダの付いていないROMに適用して下さい。

　　このパッチで変更したのは戦闘中の味方キャラクターとパレット（ガラフ侍の16色目のみ。オリジナル
　では未使用色）のみなので、データの移動がされていないものならば他のパッチにも使用できるとは思い
　ますが、使用の際は自己責任でお願いします。



				＊＊＊＊＊　パッチ内容　＊＊＊＊＊

　　戦闘中のキャラクターを、若干小顔に（FF6のキャラクタードットのように）するパッチです。
　　一部ポーズはFFRKのものを参考にしています。ドット絵は苦手なので出来の方はひどいです。
　　
　　※uosnes（20100527）で動作確認をしています。
